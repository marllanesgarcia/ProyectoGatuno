    Esta página contiene todos los elementos especificados en la rubrica.
     1. El proyecto se compone de cuatro páginas en total, más tres XML insertados en una de ellas (Colonias de Gatos).
     2. El menú, se encuentra estatico en todo momento, en la parte derecha junto a un logotipo diseñado por nosotras con mucho amor.
     3. En la página "Home" se muestra un modal explicativo (este) nada más abrir el enlace en el navegador.
     4. En la página "Colonias de Gatos" se muestra un modal explicativo (otro) nada más abrir el link.
     5. La página "Recursos" son los ajustes para aplicar cambios de color en las páginas.
     6. En la página de "Recursos", se pide un nombre de usuario, este se guarda en la base de datos para el resto de páginas; si se recarga, vuelve a pedir el nombre.
     7. "Contactanos" tiene el uso de contactar con nosotras fácilmente.
     8. Hemos usado javascript para diferentes páginas: para este modal, para enviar un gmail desde la página "Contactanos", etc.
     9. Los elementos de entrada y salida de datos sobre todo se muestran en la página de recursos.
     10. Además se muestra un mensaje por consola al inspeccionar la página.
